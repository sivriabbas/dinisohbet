# Ezan Sesi Dosyası

Bu klasöre `adhan.mp3` adında bir ezan sesi dosyası eklemeniz gerekiyor.

## Nereden Bulabilirim?

1. **Ücretsiz Ezan Sesi Kaynakları:**
   - https://www.zapsplat.com (Ücretsiz ses efektleri)
   - https://freesound.org (Ücretsiz ses kütüphanesi)
   - YouTube'dan indirme (Telif hakkına dikkat)

2. **Dosya Gereksinimleri:**
   - Dosya adı: `adhan.mp3`
   - Format: MP3
   - Önerilen süre: 1-3 dakika
   - Önerilen boyut: < 5 MB

3. **Dosya Konumu:**
   ```
   public/sounds/adhan.mp3
   ```

## Alternatif

Eğer adhan.mp3 dosyası yoksa, sistem sessiz çalışır ve sadece tarayıcı bildirimi gösterir.

## Telif Hakkı Uyarısı

Ezan sesi kullanırken telif haklarına dikkat edin. Ücretsiz ve ticari kullanıma uygun dosyalar seçin.
